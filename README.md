# 🎯 Tawjihi Smart Platform

A smart web platform designed to help Jordanian Tawjihi students understand the new education system and calculate their grades easily.

## 🚀 Features
- 📖 Full explanation of the new Tawjihi system
- 🧮 Smart grade calculators (Year 1, Year 2, Final)
- 💡 Study tips for all students
- ❓ FAQ section

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎯 Purpose
This project was built to solve a real problem faced by students in Jordan.
